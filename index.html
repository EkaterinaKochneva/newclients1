1. Анализ конкурентов по тематике
Цель – выявить лидеров, которые показывают рост посещаемости, проанализировать их и выписать себе необходимые данные для размещения на своем сайте
  • Всегда ищем конкурентов по Москве
  • Ищем молодые домены, которые показывают хороший рост
  • Фиксируем статистические показатели — возраст домена, количество страниц, трафик
  • Из общего списка 20 конкурентов выделяем 2-3 лидера, которые выделяются из общей массы
  • На основе лидеров готовим план продвижения
  • Выделяем наличие коммерческих блоков необходимых для продвижения каждой страницы
2. Чек-лист
Цель – не пропустить самые детский ошибки, которые встречаются на 80% сайтов, которые приходят в нашу компанию
  • Наш чек лист состоит из 112 параметров оптимизации
  • По каждой ошибки готовим ТЗ и выделяемся специалиста - SEO, программист, контент-менеджер, дизайнер
  • 95% ошибок исправляется в первые 2 месяца. К этому практически не возвращаемся. Ни один конкурент в будущем не сможет прислать автоматический аудита ошибками. Мы все исправили
3. Проработка структуры сайта
Цель – собрать самую широкую структуру сайта и запросов в данной тематике, чтоб в будущем было понятно за счет чего можно обойти конкурентов и что они пропустили при текущей работе
  • На основе конкурентов лидеров проработка гибридной (расширенной) структуры сайта (расположение категорий и подкатегорий сайта)
  • На базе основной (черновой) семантики расширить её и дополнить/скорректировать структуру сайта (на этом этапе будет готова структура сайта полностью)
  • Расширить семантику максимально для продвижения по трафику (найти и оставить ключевые запросы, которые несут частотность, необходимую для продвижения). После этого этапа будет готова полная семантика для сайта
4. Подготовка к оптимизации
Цель – сразу проработать основные страницы продвижения, чтоб на следующем этапе сразу внедрить все на сайте и после этого смотреть, как к нашим изменениям отнесутся поисковые системы
  • Прописать для каждой страницы сайта мета-теги (title, description, h1). После этого этапа можно внедрять семантику на сайт, переписывая основные теги
  • Сбор LSI слов на основе конкурентов-лидеров и ТОП-10 выдачи по основный (самым частотным) запросам категорий сайта
  • Проработка LSI слов и подготовка активных элементов, необходимых для внедрения на сайте
  • Доработка сайта для внедрения активных элементов (добавляются отдельные блоки/ссылки/фильтрация и т.д.)
5. Юзабилити, поведенческий фактор
Цель – сайт должен начать продавать и генерить в несколько раз больше заявок и звонков
  • Ежемесячно мы делаем 3-5 визуальных обновления на сайте
  • Внедряем новые блоки на сайте, баннеры, калькуляторы, подбираем более живые картинки, отзывы, выполненные работ, функциональные элементов и многое другое
  • Работаем с показателем отказов
  • Внедряем на сайте коммерческие страницы, чтоб сайт стал более богатым в глазах клиента
6. Статьи и копирайтинг
Цель – помочь клиенту сделать правильный выбор с помощью хорошего и профессионального текста, написанного копирайтеров совместно со специалистов в области
  • Проработка информационной части. Часть информационной семантики была собрана на этапах 1.3-1.5 остальную часть парсим с конкурентов-лидеров. Так получаем черновую информационную семантику
  • На основе семантики подбираем темы информационных статей, готовим контент план
  • Составляем ТЗ для копирайтеров + внедряем SEO оптимизированный контент
  • Оптимизация информационных статей
  • Выполняем работы по внутренняя перелинковке на основные разделы
7. Подключение сервисов и аналитика
Цель – держать руку на пульсе. Контроль Вашего сайта будет ежедневно происходить с помощью 7 современных сервисов, которые мы оплачиваем
8. Регулярные работы
Цель – системно улучшать показатели оптимизации URL по запросам
  • Улучшение показателей каждой страницы и сайта, поведенческие факторы, докрутка текстового анализа, постановка гипотез, копирайтинг, анализ ТОП, анализ выдачи по поисковым запросам, повтор цикла.
9. Отчетность
KPI, Отчеты, анализ ключевых результатов. Чек листы: контент, верстка, программирование.
10. Корректировка планов, стратегии
11. Внешняя оптимизация
12. Оценка отдачи
13. Комплексные задачи
Крауд маркетинг, социальные сети(вконтакте, facebook, instagram, youtube), увеличение охватов.
14. Повтор цикла
